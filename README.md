- Colors & Backgrounds
- Typography
- Box Model
- Display Properties
- Positioning
- Flexbox
- CSS Grid
- Transitions
- Animations
- Transforms
- Pseudo-classes
- Pseudo-elements
- Responsive Design
- Media Queries
- CSS Variables
- Gradients
- Shadows
- Overflow
- Z-index
- And more...
